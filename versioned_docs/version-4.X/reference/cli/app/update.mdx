# `app update` Command

The `app update` command applies updates one or all apps.

## Usage

```bash
app update <app_name>
```

### Parameters

| Parameter  | Description                      | Required? | Default Value |
| ---------- | -------------------------------- | --------- | ------------- |
| `app_name` | `all` or infrastructure app name | ✔️        |               |

## Example

```bash
> app update all
# highlight-start
App cleanup-untagged updated successfully to the latest version (1.0.0)
# highlight-end
```
