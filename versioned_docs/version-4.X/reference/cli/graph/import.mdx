# `graph import` Command

The `graph import` command imports a Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshot from disk.

## Usage

```bash
graph import [--force] [graph_name] <file_name>
```

## Example

```bash
> graph import resoto_test fixinventory.graph
# highlight-start
Graph resoto_test imported from fixinventory.graph.
# highlight-end
```
