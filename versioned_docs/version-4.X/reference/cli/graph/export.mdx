# `graph export` Command

The `graph export` command exports a Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshot to disk.

## Usage

```bash
graph export [--force] [graph_name] <file_name>
```

## Example

```bash
> graph export fix fixinventory.graph
# highlight-start
Received a file fixinventory.graph, which is stored to ./fixinventory.graph
# highlight-end
```
