# `graph list` Command

The `graph list` command lists all Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshots.

## Example

```bash
> graph list
# highlight-start
fix
snapshot-fix-hourly-20230614T222121Z
snapshot-fix-daily-20230614T222121Z
# highlight-end
```
