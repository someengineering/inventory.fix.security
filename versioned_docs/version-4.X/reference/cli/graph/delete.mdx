# `graph delete` Command

The `graph delete` command deletes a Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshot.

## Usage

```bash
graph delete <graph_name>
```

## Example

```bash
> graph delete resoto_backup
# highlight-start
Graph resoto_backup deleted.
# highlight-end
```
