# `azure_postgresql_ad_administrator`

- Categories: Database
- Service: postgresql
- Provider Link: [PostgreSQL Ad Administrator](https://learn.microsoft.com/en-us/azure/postgresql/flexible-server/how-to-configure-sign-in-azure-ad-authentication)

## Description

Azure PostgreSQL AD Administrator is a role that manages access to Azure Database for PostgreSQL using Azure Active Directory credentials. It controls user authentication, assigns database roles, and sets permissions for AD users and groups. This administrator can create and manage logins, ensuring secure and centralized identity management for PostgreSQL databases in Azure.
