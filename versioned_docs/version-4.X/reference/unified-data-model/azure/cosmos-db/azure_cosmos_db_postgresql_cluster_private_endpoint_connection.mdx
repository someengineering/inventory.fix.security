# `azure_cosmos_db_postgresql_cluster_private_endpoint_connection`

- Categories: Networking
- Service: cosmos-db
- Provider Link: [Cosmos DB PostgreSQL Cluster Private Endpoint Connection](https://learn.microsoft.com/en-us/azure/cosmos-db/postgresql/how-to-configure-private-endpoints)

## Description

Azure Cosmos DB PostgreSQL Cluster Private Endpoint Connection provides a secure, private network link between a virtual network and an Azure Cosmos DB PostgreSQL cluster. It uses Azure Private Link to create this connection, bypassing the public internet. This feature enhances data security and reduces exposure to potential threats while maintaining network performance.
