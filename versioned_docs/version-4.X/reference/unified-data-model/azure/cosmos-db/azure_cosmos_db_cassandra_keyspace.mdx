# `azure_cosmos_db_cassandra_keyspace`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Cassandra Keyspace](https://learn.microsoft.com/en-us/azure/cosmos-db/cassandra/)

## Description

Azure Cosmos DB Cassandra Keyspace is a container within Azure Cosmos DB that organizes data for Apache Cassandra workloads. It functions as a namespace for tables and provides a logical grouping for related data. Users can create, manage, and query keyspaces using Cassandra Query Language (CQL), maintaining compatibility with existing Cassandra applications.
