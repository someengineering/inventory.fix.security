# `azure_cosmos_db_sql_role_definition`

- Categories: Access Control
- Service: cosmos-db
- Provider Link: [Cosmos DB SQL Role Definition](https://learn.microsoft.com/en-us/azure/cosmos-db/sql/how-to-setup-rbac)

## Description

Azure Cosmos DB SQL Role Definition is a security feature that defines permissions for users and applications accessing Cosmos DB resources. It specifies the actions allowed on containers, databases, and items within a Cosmos DB account. Administrators can create custom roles to manage access control and implement principle of least privilege in their database environments.
