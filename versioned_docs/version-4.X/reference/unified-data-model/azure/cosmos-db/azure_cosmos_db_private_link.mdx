# `azure_cosmos_db_private_link`

- Categories: Networking
- Service: cosmos-db
- Provider Link: [Cosmos DB Private Link](https://learn.microsoft.com/en-us/azure/cosmos-db/how-to-configure-private-endpoints)

## Description

Azure Cosmos DB Private Link is a network security feature that provides private connectivity to Cosmos DB accounts. It creates a direct connection between virtual networks and Cosmos DB resources using Microsoft's private network infrastructure. This service restricts access to specific resources, enhancing data isolation and reducing exposure to public internet threats.
