# `azure_cosmos_db_restorable_sql_container`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Restorable SQL Container](https://learn.microsoft.com/en-us/azure/cosmos-db/sql/restorable-container-properties)

## Description

Azure Cosmos DB Restorable SQL Container is a feature that provides point-in-time recovery for SQL API containers in Azure Cosmos DB. It creates continuous backups of container data, letting users restore to any timestamp within the retention period. This functionality helps protect against accidental deletions, modifications, or corruption of data.
