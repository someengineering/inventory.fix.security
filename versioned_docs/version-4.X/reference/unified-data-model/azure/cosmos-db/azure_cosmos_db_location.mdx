# `azure_cosmos_db_location`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Location](https://learn.microsoft.com/en-us/azure/cosmos-db/location-based-routing)

## Description

Azure Cosmos DB Location refers to the geographic region where an Azure Cosmos DB database is deployed and hosted. It determines the physical data center that stores and processes the database, affecting latency, compliance, and data residency. Users can select specific locations to optimize performance and meet regulatory requirements for their applications.
