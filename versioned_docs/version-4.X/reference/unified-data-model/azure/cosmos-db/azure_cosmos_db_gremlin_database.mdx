# `azure_cosmos_db_gremlin_database`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Gremlin Database](https://learn.microsoft.com/en-us/azure/cosmos-db/gremlin/)

## Description

Azure Cosmos DB Gremlin Database is a graph database service within Microsoft's Azure cloud platform. It supports the Apache TinkerPop Gremlin API for storing and querying graph data. Users can model complex relationships between entities, perform traversals, and execute graph operations. The service offers global distribution and multi-region writes for graph data workloads.
