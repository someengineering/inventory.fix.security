# `azure_cosmos_db_mongo_db_user_definition`

- Categories: Access Control
- Service: cosmos-db
- Provider Link: [Cosmos DB Mongo DB User Definition](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/)

## Description

Azure Cosmos DB MongoDB User Definition specifies access permissions for users in a MongoDB database within Azure Cosmos DB. It defines authentication credentials and authorization roles, controlling database operations a user can perform. This configuration helps manage security and access control for MongoDB workloads in the Azure Cosmos DB environment.
