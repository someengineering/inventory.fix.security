# `azure_cosmos_db_mongo_db_cluster`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Mongo DB Cluster](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/)

## Description

Azure Cosmos DB Mongo DB Cluster is a managed database service that provides MongoDB compatibility on Microsoft's Azure cloud platform. It offers distributed data storage and retrieval, supporting MongoDB APIs and drivers. Users can deploy, scale, and manage MongoDB databases while benefiting from Azure's global infrastructure, automatic backups, and security features.
