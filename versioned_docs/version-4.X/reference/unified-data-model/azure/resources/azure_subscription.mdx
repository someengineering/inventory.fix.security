# `azure_subscription`

- Categories: Management
- Service: resources
- Provider Link: [Subscription](https://learn.microsoft.com/en-us/azure/cost-management-billing/manage/create-subscription)

## Description

An Azure Subscription is a logical container for organizing and managing Microsoft Azure resources. It provides access to cloud services and defines usage limits and billing arrangements. Users can create, deploy, and control Azure resources within their subscription, while Microsoft tracks resource consumption and generates invoices based on the subscription's payment model.
