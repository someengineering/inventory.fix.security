# `azure_sql_server_managed_instance_ad_administrator`

- Categories: Access Control
- Service: azure-sql
- Provider Link: [SQL Server Managed Instance Ad Administrator](https://learn.microsoft.com/en-us/azure/azure-sql/managed-instance/authentication-aad-configure?view=azuresql#provision-azure-ad-administrator)

## Description

Azure SQL Server Managed Instance AD Administrator is a role that manages authentication and authorization for a SQL Managed Instance using Azure Active Directory. It configures and maintains AD integration, sets up single sign-on, and controls access to database resources. This role ensures secure user authentication and simplifies identity management within the SQL Managed Instance environment.
