# `azure_sql_server_managed_instance`

- Categories: Database
- Service: azure-sql
- Provider Link: [SQL Server Managed Instance](https://learn.microsoft.com/en-us/azure/azure-sql/managed-instance/)

## Description

Azure SQL Server Managed Instance is a cloud database service that provides SQL Server compatibility with automated management features. It offers a near-complete parity with on-premises SQL Server instances, including native virtual network support and full SQL Server engine compatibility, while handling routine database management tasks such as backups, patching, and monitoring.
