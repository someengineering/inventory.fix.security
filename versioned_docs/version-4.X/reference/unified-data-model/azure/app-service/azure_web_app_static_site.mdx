# `azure_web_app_static_site`

- Categories: Compute
- Service: app-service
- Provider Link: [Web App Static Site](https://learn.microsoft.com/en-us/azure/static-web-apps/)

## Description

Azure Web App Static Site is a service for hosting static web content. It offers global content distribution, automated builds and deployments from code repositories, and integration with Azure services. Users can deploy HTML, CSS, JavaScript, and image files directly from their repositories, with support for custom domains and free SSL certificates.
