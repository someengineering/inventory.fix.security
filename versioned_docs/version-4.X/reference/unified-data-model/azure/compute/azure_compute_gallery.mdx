# `azure_compute_gallery`

- Categories: Compute
- Service: compute
- Provider Link: [Compute Gallery](https://learn.microsoft.com/en-us/azure/virtual-machines/shared-image-galleries)

## Description

Azure Compute Gallery is a service for storing and managing virtual machine images and related artifacts. It provides a central repository for sharing, versioning, and replicating images across Azure regions and subscriptions. Users can create and maintain custom images, distribute them to different teams, and deploy virtual machines from these shared resources.
