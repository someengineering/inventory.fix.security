# `azure_compute_disk_type`

- Categories: Storage
- Service: compute
- Provider Link: [Compute Disk Type](https://learn.microsoft.com/en-us/azure/virtual-machines/disks-types)

## Description

Azure Compute Disk Types are storage options for virtual machines in Microsoft Azure. They provide persistent block storage for data and applications. Users can choose from different disk types, including standard HDD, standard SSD, and premium SSD, each offering varying performance levels and capacities to meet specific workload requirements and budget constraints.
