# `azure_monitor_diagnostic_settings`

- Categories: Management
- Service: monitor
- Provider Link: [Monitor Diagnostic Settings](https://learn.microsoft.com/en-us/azure/azure-monitor/essentials/diagnostic-settings)

## Description

Azure Monitor Diagnostic Settings is a feature that collects and routes platform logs and metrics from Azure resources to specified destinations. It supports sending data to Log Analytics workspaces, storage accounts, and event hubs. Users can configure multiple settings per resource, selecting specific log categories and metrics to capture, aiding in monitoring and analysis of Azure resources.
