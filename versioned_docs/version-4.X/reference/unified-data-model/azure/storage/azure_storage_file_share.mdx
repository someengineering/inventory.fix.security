# `azure_storage_file_share`

- Categories: Storage
- Service: storage
- Provider Link: [Storage File Share](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction)

## Description

Azure Storage File Share is a cloud-based file storage service provided by Microsoft Azure. It offers shared access to files across multiple virtual machines and applications using the Server Message Block (SMB) protocol. Users can mount file shares on Windows, Linux, and macOS systems, and access them like traditional network drives.
