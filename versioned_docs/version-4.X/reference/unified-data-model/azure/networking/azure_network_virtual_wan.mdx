# `azure_network_virtual_wan`

- Categories: Networking
- Service: networking
- Provider Link: [Network Virtual WAN](https://learn.microsoft.com/en-us/azure/virtual-wan/)

## Description

Azure Network Virtual WAN is a Microsoft networking service that connects and manages multiple networks across different locations. It integrates on-premises networks, branch offices, and remote sites with Azure virtual networks. The service provides automated routing, optimized data paths, and simplified connectivity for organizations with distributed network infrastructures, supporting both site-to-site and point-to-site VPN configurations.
