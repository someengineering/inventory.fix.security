# `azure_network_vpn_server_configuration`

- Categories: Networking
- Service: networking
- Provider Link: [Network VPN Server Configuration](https://learn.microsoft.com/en-us/azure/vpn-gateway/vpn-gateway-about-vpn-devices)

## Description

Azure Network VPN Server Configuration is a service in Microsoft Azure that sets up and manages virtual private network gateways. It creates secure connections between on-premises networks and Azure virtual networks, encrypting data transmitted over the internet. This configuration supports site-to-site, point-to-site, and VNet-to-VNet connections, providing remote access and network integration capabilities.
