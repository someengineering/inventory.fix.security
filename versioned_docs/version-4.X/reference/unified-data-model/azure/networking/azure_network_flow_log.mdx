# `azure_network_flow_log`

- Categories: Networking
- Service: networking
- Provider Link: [Network Flow Log](https://learn.microsoft.com/en-us/azure/network-watcher/network-watcher-nsg-flow-logging-overview)

## Description

Azure Network Flow Log is a feature that records information about IP traffic flowing through Azure network security groups. It captures data on allowed and denied connections, including source and destination IP addresses, ports, protocols, and traffic direction. This log helps administrators monitor network activity, troubleshoot connectivity issues, and enhance security posture.
