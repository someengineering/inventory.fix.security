# `azure_network_express_route_cross_connection`

- Categories: Networking
- Service: networking
- Provider Link: [Network Express Route Cross Connection](https://learn.microsoft.com/en-us/azure/expressroute/expressroute-cross-connections-overview)

## Description

Azure Network Express Route Cross Connection is a service that links on-premises networks to Microsoft Azure through a dedicated private connection. It bypasses the public internet, providing direct access to Azure services with increased reliability, lower latency, and higher security. This connection supports various network protocols and can be used for data transfer, application hosting, and disaster recovery.
