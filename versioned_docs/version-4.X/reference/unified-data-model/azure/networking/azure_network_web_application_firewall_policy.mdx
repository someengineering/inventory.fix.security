# `azure_network_web_application_firewall_policy`

- Categories: Networking
- Service: networking
- Provider Link: [Network Web Application Firewall Policy](https://learn.microsoft.com/en-us/azure/web-application-firewall/ag/policy-overview)

## Description

Azure Network Web Application Firewall Policy is a security configuration for web applications in Azure. It defines rules to protect against common web vulnerabilities and threats. The policy filters incoming traffic, blocks malicious requests, and enforces security measures. It can be applied to Azure Application Gateway, Azure Front Door, and Azure CDN to safeguard web applications.
