# `azure_machine_learning_compute`

- Categories: Compute
- Service: machine-learning
- Provider Link: [Machine Learning Compute](https://docs.microsoft.com/en-us/azure/machine-learning/concept-compute-target)

## Description

Azure Machine Learning Compute is a cloud-based service for running machine learning workloads. It provides managed compute resources for training and deploying models. Users can create and manage compute clusters, select virtual machine sizes, and autoscale resources as needed. The service supports various machine learning frameworks and integrates with other Azure services for data processing and model deployment.
