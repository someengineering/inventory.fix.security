# `azure_machine_learning_registry`

- Categories: Machine Learning & AI
- Service: machine-learning
- Provider Link: [Machine Learning Registry](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-use-model-registry)

## Description

Azure Machine Learning Registry is a centralized repository for storing and managing machine learning models, datasets, and components. It provides version control, collaboration features, and integration with Azure Machine Learning workflows. Users can publish, share, and deploy models across their organization, ensuring reproducibility and governance in machine learning projects.
