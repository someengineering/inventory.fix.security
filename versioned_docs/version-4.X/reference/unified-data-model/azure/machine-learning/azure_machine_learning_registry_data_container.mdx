# `azure_machine_learning_registry_data_container`

- Categories: Machine Learning & AI
- Service: machine-learning
- Provider Link: [Machine Learning Registry Data Container](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-share-models-pipelines-across-workspaces-with-registries)

## Description

Azure Machine Learning Registry Data Container is a storage solution for machine learning artifacts in Azure. It stores and manages models, datasets, and environments, providing version control and collaboration features. Users can access, share, and deploy these artifacts across projects and teams, supporting the entire machine learning lifecycle within the Azure ecosystem.
