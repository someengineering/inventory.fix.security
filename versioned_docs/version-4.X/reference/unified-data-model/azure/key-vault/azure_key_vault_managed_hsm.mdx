# `azure_key_vault_managed_hsm`

- Categories: Management
- Service: key-vault
- Provider Link: [Key Vault Managed Hsm](https://learn.microsoft.com/en-us/azure/key-vault/managed-hsm/)

## Description

Azure Key Vault Managed HSM is a cloud service that provides secure storage and management of cryptographic keys and secrets. It uses FIPS 140-2 Level 3 validated hardware security modules (HSMs) to protect sensitive data. Users can create, store, and control access to encryption keys, passwords, and certificates within the Azure cloud environment.
