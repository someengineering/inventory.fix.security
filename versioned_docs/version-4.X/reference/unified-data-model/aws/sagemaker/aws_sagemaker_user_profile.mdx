# `aws_sagemaker_user_profile`

- Categories: Machine Learning & AI
- Service: sagemaker
- Provider Link: [SageMaker User Profile](https://docs.aws.amazon.com/sagemaker/latest/dg/domain-user-profile.html)

## Description

AWS SageMaker User Profile is a feature within Amazon SageMaker that manages individual user settings and permissions in a SageMaker Domain. It stores user-specific configurations, including execution roles, security groups, and custom settings. User Profiles control access to SageMaker resources and define user-specific environments, ensuring proper isolation and resource allocation for machine learning projects.
