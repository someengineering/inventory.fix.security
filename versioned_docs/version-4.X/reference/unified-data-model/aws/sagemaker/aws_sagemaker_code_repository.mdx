# `aws_sagemaker_code_repository`

- Categories: Machine Learning & AI
- Service: sagemaker
- Provider Link: [SageMaker Code Repository](https://docs.aws.amazon.com/sagemaker/latest/dg/nbi-git-repo.html)

## Description

AWS SageMaker Code Repository is a version control service integrated with SageMaker. It hosts Git repositories for machine learning projects, providing a centralized location for storing and managing code. Users can access, edit, and version their code directly within SageMaker, facilitating collaboration and maintaining a history of changes throughout the development process.
