# `aws_sagemaker_training_job`

- Categories: Machine Learning & AI
- Service: sagemaker
- Provider Link: [SageMaker Training Job](https://docs.aws.amazon.com/sagemaker/latest/dg/train-model.html)

## Description

AWS SageMaker Training Job is a service that trains machine learning models using specified algorithms and datasets. It provisions and manages the necessary compute resources, executes the training process, and stores the resulting model artifacts. Users can configure training parameters, input data sources, and output locations. The service supports various machine learning frameworks and algorithms.
