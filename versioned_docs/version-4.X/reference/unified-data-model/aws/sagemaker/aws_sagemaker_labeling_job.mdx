# `aws_sagemaker_labeling_job`

- Categories: Machine Learning & AI
- Service: sagemaker
- Provider Link: [SageMaker Labeling Job](https://docs.aws.amazon.com/sagemaker/latest/dg/sms.html)

## Description

AWS SageMaker Labeling Job is a feature within Amazon SageMaker that facilitates data labeling for machine learning projects. It provides tools for creating, managing, and monitoring labeling tasks. Users can assign work to human annotators, track progress, and review results. The service supports various data types including images, text, and audio for classification and object detection tasks.
