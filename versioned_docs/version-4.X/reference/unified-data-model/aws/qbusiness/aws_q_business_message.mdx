# `aws_q_business_message`

- Categories: Machine Learning & AI
- Service: qbusiness
- Provider Link: [Q Business Message](https://docs.aws.amazon.com/qbusiness/latest/adminguide/what-is-qbusiness.html)

## Description

AWS Q Business Message is a generative AI-powered tool for business communication within Amazon Web Services. It helps users create professional messages, emails, and documents by understanding context and generating appropriate content. The service aims to improve communication efficiency and quality while maintaining consistency with company standards and guidelines.
