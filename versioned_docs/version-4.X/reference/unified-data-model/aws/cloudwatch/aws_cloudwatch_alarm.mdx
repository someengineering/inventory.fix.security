# `aws_cloudwatch_alarm`

- Categories: Management
- Service: cloudwatch
- Provider Link: [CloudWatch Alarm](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/AlarmThatSendsEmail.html)

## Description

AWS CloudWatch Alarm is a monitoring service that tracks metrics from AWS resources and applications. It sends notifications or performs automated actions when predefined thresholds are breached. Users can set alarms for various metrics, including CPU utilization, network traffic, and custom application metrics. CloudWatch Alarm integrates with other AWS services to trigger responses based on specified conditions.
