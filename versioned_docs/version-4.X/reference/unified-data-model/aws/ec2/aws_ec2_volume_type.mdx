# `aws_ec2_volume_type`

- Categories: Storage
- Service: ec2
- Provider Link: [EC2 Volume Type](https://docs.aws.amazon.com/)

## Description

EC2 Volume Types are different storage options for Amazon Elastic Block Store (EBS) volumes, such as General Purpose (SSD) and Magnetic.
