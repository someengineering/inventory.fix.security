# `aws_ec2_keypair`

- Categories: Access Control
- Service: ec2
- Provider Link: [EC2 Keypair](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/ec2-key-pairs.html)

## Description

AWS EC2 Keypair is a security credential for Amazon Elastic Compute Cloud. It consists of a public key stored by AWS and a private key file kept by the user. This keypair authenticates secure SSH connections to EC2 instances, providing access control and encryption for remote management of virtual servers in the AWS cloud environment.
