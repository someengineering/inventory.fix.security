# `aws_autoscaling_group`

- Categories: Compute, Management
- Service: autoscaling
- Provider Link: [Autoscaling Group](https://docs.aws.amazon.com/autoscaling/ec2/userguide/auto-scaling-groups.html)

## Description

AWS Auto Scaling Group is a service that manages multiple EC2 instances as a collective unit. It automatically adjusts the number of instances based on defined conditions, ensuring optimal performance and resource utilization. It maintains a specified number of instances, replaces unhealthy ones, and scales capacity up or down according to demand or schedule.
