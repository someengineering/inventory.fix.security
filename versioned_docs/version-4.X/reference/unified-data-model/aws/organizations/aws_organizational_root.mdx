# `aws_organizational_root`

- Categories: Management
- Service: organizations
- Provider Link: [Organizational Root](https://docs.aws.amazon.com/organizations/latest/userguide/orgs_manage_org_root.html)

## Description

AWS Organizational Root is the top-level entity in AWS Organizations. It serves as the starting point for creating and managing multiple AWS accounts within an organization. The root provides centralized control over billing, access management, and resource allocation across all member accounts, ensuring consistent policies and governance throughout the organizational structure.
