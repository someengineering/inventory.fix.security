# `aws_sns_endpoint`

- Categories: Networking
- Service: sns
- Provider Link: [SNS Endpoint](https://docs.aws.amazon.com/sns/latest/dg/sns-create-endpoint.html)

## Description

AWS SNS Endpoint is a destination for Amazon Simple Notification Service (SNS) messages. It represents a specific target where notifications are sent, such as mobile devices, email addresses, or web servers. SNS Endpoints receive messages published to SNS topics, facilitating the distribution of information across various platforms and applications within AWS infrastructure.
