# `aws_sns_subscription`

- Categories: Networking
- Service: sns
- Provider Link: [SNS Subscription](https://docs.aws.amazon.com/sns/latest/dg/sns-create-subscribe-endpoint-to-topic.html)

## Description

AWS SNS Subscription is a feature of Amazon Simple Notification Service that lets users receive messages from SNS topics. It supports multiple protocols, including email, SMS, HTTP/S, and AWS Lambda. Subscribers can filter messages based on attributes, ensuring they only receive relevant information. SNS Subscription facilitates message delivery across distributed systems and applications.
