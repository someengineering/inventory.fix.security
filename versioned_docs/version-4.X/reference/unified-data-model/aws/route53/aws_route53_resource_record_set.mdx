# `aws_route53_resource_record_set`

- Categories: DNS, Networking
- Service: route53
- Provider Link: [Route53 Resource Record Set](https://docs.aws.amazon.com/Route53/latest/APIReference/API_ResourceRecordSet.html)

## Description

AWS Route 53 Resource Record Set is a collection of DNS records for a domain. It defines how traffic is routed to resources like websites, email servers, or other services. These records include types such as A, AAAA, CNAME, MX, and TXT, each serving specific purposes in DNS resolution and domain configuration.
