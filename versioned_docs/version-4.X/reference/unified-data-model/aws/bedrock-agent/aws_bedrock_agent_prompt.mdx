# `aws_bedrock_agent_prompt`

- Categories: Machine Learning & AI
- Service: bedrock-agent
- Provider Link: [Bedrock Agent Prompt](https://docs.aws.amazon.com/bedrock/latest/userguide/agents-prompt.html)

## Description

AWS Bedrock Agent Prompt is a feature that helps developers create AI agents within the AWS Bedrock service. It provides a framework for defining agent behaviors, integrating with foundation models, and handling user interactions. The prompt system guides the creation of agents that can perform tasks, answer questions, and assist users in various applications.
