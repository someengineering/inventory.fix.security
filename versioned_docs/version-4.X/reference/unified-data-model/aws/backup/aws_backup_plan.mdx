# `aws_backup_plan`

- Categories: Storage
- Service: backup
- Provider Link: [Backup Plan](https://docs.aws.amazon.com/aws-backup/latest/devguide/create-a-backup-plan.html)

## Description

AWS Backup Plan is a feature of AWS Backup that automates and manages data protection for AWS resources. It defines when and how often backups are created, specifies retention periods for backups, and sets rules for copying backups across AWS Regions or accounts. Users can create custom backup plans or use pre-configured plans to meet their data protection requirements.
