# `aws_backup_legal_hold`

- Categories: Storage
- Service: backup
- Provider Link: [Backup Legal Hold](https://docs.aws.amazon.com/aws-backup/latest/devguide/legal-holds.html)

## Description

AWS Backup Legal Hold is a feature that preserves backup copies of data for legal or compliance purposes. It prevents the deletion or modification of specified backups, ensuring data remains intact and accessible during investigations or litigation. Users can apply legal holds to backups across multiple AWS services, maintaining data integrity and meeting regulatory requirements.
