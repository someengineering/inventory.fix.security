# `aws_backup_recovery_point`

- Categories: Storage
- Service: backup
- Provider Link: [Backup Recovery Point](https://docs.aws.amazon.com/aws-backup/latest/devguide/recovery-points.html)

## Description

AWS Backup Recovery Point is a snapshot of data created by AWS Backup. It represents a point-in-time copy of resources, such as EBS volumes, RDS databases, or EFS file systems. Recovery Points can be used to restore data to a specific state, helping organizations recover from data loss or system failures.
