# `aws_backup_job`

- Categories: Storage
- Service: backup
- Provider Link: [Backup Job](https://docs.aws.amazon.com/aws-backup/latest/devguide/backup-jobs.html)

## Description

AWS Backup Job is a task that creates and manages backups of AWS resources. It defines the source data, backup frequency, and retention policy. The job executes according to a schedule, creating point-in-time snapshots of specified resources. It handles backup creation, storage, and lifecycle management, ensuring data protection and recovery capabilities for AWS environments.
