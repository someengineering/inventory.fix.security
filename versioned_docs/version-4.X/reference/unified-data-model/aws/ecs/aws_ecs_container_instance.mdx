# `aws_ecs_container_instance`

- Categories: Compute
- Service: ecs
- Provider Link: [ECS Container Instance](https://docs.aws.amazon.com/AmazonECS/latest/developerguide/ECS_instances.html)

## Description

An AWS ECS Container Instance is a virtual machine that runs the Amazon ECS container agent and Docker daemon. It serves as a host for containers managed by Amazon ECS. These instances provide compute resources for running containers, handle container scheduling, and communicate with the ECS service to manage tasks and report instance status.
