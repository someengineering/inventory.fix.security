# `aws_bedrock_evaluation_job`

- Categories: Machine Learning & AI, Compute
- Service: bedrock
- Provider Link: [Bedrock Evaluation Job](https://docs.aws.amazon.com/bedrock/latest/userguide/evaluation-jobs.html)

## Description

AWS Bedrock Evaluation Job is a feature that assesses the performance of foundation models in AWS Bedrock. It runs a set of predefined or custom tasks on selected models, comparing their outputs against human-generated responses. The job generates metrics and reports to help users evaluate model quality and suitability for specific use cases.
