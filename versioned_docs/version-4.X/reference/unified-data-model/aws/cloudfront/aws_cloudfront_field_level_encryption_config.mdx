# `aws_cloudfront_field_level_encryption_config`

- Categories: Networking
- Service: cloudfront
- Provider Link: [CloudFront Field-Level Encryption Configuration](https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/field-level-encryption.html)

## Description

AWS CloudFront Field-Level Encryption Configuration is a security feature that encrypts sensitive data fields in web forms before transmission to origin servers. It uses public key cryptography to protect specific information, ensuring that only authorized applications with the corresponding private key can decrypt the data, enhancing security for sensitive user input during transit.
