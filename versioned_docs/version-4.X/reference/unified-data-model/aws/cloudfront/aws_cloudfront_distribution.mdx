# `aws_cloudfront_distribution`

- Categories: Networking
- Service: cloudfront
- Provider Link: [CloudFront Distribution](https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/distribution-working-with.html)

## Description

AWS CloudFront Distribution is a content delivery network service that caches and serves web content from edge locations worldwide. It reduces latency by delivering data, videos, applications, and APIs to users from the nearest server. CloudFront integrates with other AWS services and supports custom domain names, SSL certificates, and security features like DDoS protection.
