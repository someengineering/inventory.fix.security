# `aws_rds_cluster`

- Categories: Compute, Database
- Service: rds
- Provider Link: [RDS Cluster](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/Aurora.Overview.html)

## Description

AWS RDS Cluster is a managed database service that groups multiple database instances together. It provides high availability and fault tolerance by replicating data across instances in different availability zones. Users can scale read capacity by adding read replicas and perform automatic failover to maintain database accessibility during instance failures or maintenance.
