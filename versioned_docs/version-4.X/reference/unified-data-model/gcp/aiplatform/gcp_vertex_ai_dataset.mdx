# `gcp_vertex_ai_dataset`

- Categories: Machine Learning & AI
- Service: aiplatform
- Provider Link: [Vertex AI Dataset](https://cloud.google.com/docs)

## Description

A Dataset is a container for a collection of data used to train and evaluate machine learning models in Vertex AI. Each dataset contains metadata about the data source and structure.
