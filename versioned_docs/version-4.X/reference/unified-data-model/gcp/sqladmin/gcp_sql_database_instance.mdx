# `gcp_sql_database_instance`

- Categories: Compute, Database
- Service: sqladmin
- Provider Link: [SQL Database Instance](https://cloud.google.com/sql/docs/mysql/instance-settings)

## Description

GCP SQL Database Instance is a managed relational database service on Google Cloud Platform. It provides MySQL, PostgreSQL, and SQL Server databases with automated backups, replication, and patching. Users can create, configure, and operate database instances without managing the underlying infrastructure, focusing on application development and data management tasks.
