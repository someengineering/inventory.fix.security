# `gcp_sql_operation`

- Categories: Database
- Service: sqladmin
- Provider Link: [SQL Operation](https://cloud.google.com/sql/docs/mysql/admin-api/rest/v1/operations)

## Description

GCP SQL Operation represents an asynchronous database operation in Google Cloud Platform's Cloud SQL service. It tracks the progress and status of tasks like instance creation, modification, or deletion. Users can monitor, manage, and retrieve information about these operations through the Cloud SQL API or command-line interface, ensuring visibility into database management processes.
