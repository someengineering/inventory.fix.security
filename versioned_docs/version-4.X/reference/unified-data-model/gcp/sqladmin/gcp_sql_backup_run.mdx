# `gcp_sql_backup_run`

- Categories: Database
- Service: sqladmin
- Provider Link: [SQL Backup Run](https://cloud.google.com/sql/docs/mysql/backup-recovery/backups)

## Description

GCP SQL Backup Run is a feature of Google Cloud Platform that creates and manages backups for Cloud SQL databases. It automatically captures database snapshots at scheduled intervals, storing them securely in Google Cloud Storage. These backups can be used to restore databases to specific points in time, protecting against data loss and facilitating disaster recovery.
