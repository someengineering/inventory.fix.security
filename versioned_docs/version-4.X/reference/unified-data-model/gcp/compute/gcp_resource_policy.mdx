# `gcp_resource_policy`

- Categories: Management
- Service: compute
- Provider Link: [Resource Policy](https://cloud.google.com/compute/docs/reference/rest/v1/resourcePolicies)

## Description

GCP Resource Policy is a Google Cloud Platform feature that defines rules and constraints for resource management. It controls the creation, modification, and deletion of resources across projects and organizations. Resource policies enforce governance, security, and compliance standards, ensuring consistent resource configurations and reducing potential misconfigurations or unauthorized actions within a GCP environment.
