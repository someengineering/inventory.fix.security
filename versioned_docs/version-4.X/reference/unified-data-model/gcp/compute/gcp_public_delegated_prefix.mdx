# `gcp_public_delegated_prefix`

- Categories: Networking
- Service: compute
- Provider Link: [Public Delegated Prefix](https://cloud.google.com/network-connectivity/docs/network-connectivity-center/concepts/public-delegated-prefix)

## Description

GCP Public Delegated Prefix is a networking feature that lets users bring their own IP addresses to Google Cloud. It assigns a specific range of public IP addresses to a project, which can be used for various resources within that project. This feature provides control over IP address management and facilitates migration of existing workloads to GCP.
