# `gcp_health_check_service`

- Categories: Access Control
- Service: compute
- Provider Link: [Health Check Service](https://cloud.google.com/load-balancing/docs/health-checks)

## Description

GCP Health Check Service monitors the health and performance of applications and services running on Google Cloud Platform. It periodically sends probes to specified endpoints, verifying their availability and responsiveness. The service can detect issues, trigger alerts, and integrate with load balancers to route traffic away from unhealthy instances, helping maintain application reliability and uptime.
