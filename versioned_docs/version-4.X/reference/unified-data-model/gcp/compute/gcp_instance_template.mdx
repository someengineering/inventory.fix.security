# `gcp_instance_template`

- Categories: Compute
- Service: compute
- Provider Link: [Instance Template](https://cloud.google.com/compute/docs/instance-templates)

## Description

A GCP Instance Template is a resource that defines the configuration for virtual machine instances in Google Cloud Platform. It specifies the machine type, boot disk image, network interfaces, and other settings. Instance Templates serve as blueprints for creating multiple identical instances or instance groups, facilitating consistent and repeatable VM deployments across projects.
