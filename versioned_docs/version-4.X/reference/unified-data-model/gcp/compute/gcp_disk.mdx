# `gcp_disk`

- Categories: Storage
- Service: compute
- Provider Link: [Disk](https://cloud.google.com/compute/docs/disks)

## Description

GCP Disk is a storage option in Google Cloud Platform that provides persistent block storage for virtual machines. It functions as a hard drive for compute instances, storing data and operating systems. GCP Disk offers various types, including standard, balanced, and SSD, with different performance characteristics to suit diverse workload requirements.
