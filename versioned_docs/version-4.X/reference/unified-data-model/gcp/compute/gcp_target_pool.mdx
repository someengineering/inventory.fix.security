# `gcp_target_pool`

- Categories: Networking
- Service: compute
- Provider Link: [Target Pool](https://cloud.google.com/load-balancing/docs/target-pools)

## Description

A GCP Target Pool is a network resource that distributes incoming traffic across multiple instances in a region. It acts as a single endpoint for load balancing, directing requests to healthy instances within the pool. Target Pools can be used with network load balancers to distribute traffic based on IP protocol data.
