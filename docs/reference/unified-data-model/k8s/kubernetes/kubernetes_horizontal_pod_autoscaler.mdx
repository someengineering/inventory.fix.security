# `kubernetes_horizontal_pod_autoscaler`

- Categories: Other
- Service: kubernetes

## Description

The Kubernetes Horizontal Pod Autoscaler automatically scales the number of pods in a deployment up or down based on CPU usage or other specified metrics.
