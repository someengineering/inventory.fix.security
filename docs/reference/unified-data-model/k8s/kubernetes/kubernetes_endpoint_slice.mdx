# `kubernetes_endpoint_slice`

- Categories: Other
- Service: kubernetes

## Description

Kubernetes Endpoint Slices are a feature that allows for more efficient and scalable service discovery in a Kubernetes cluster by splitting endpoints into smaller slices.
