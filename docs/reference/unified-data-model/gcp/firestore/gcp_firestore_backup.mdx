# `gcp_firestore_backup`

- Categories: Storage
- Service: firestore
- Provider Link: [Firestore Backup](https://cloud.google.com/docs)

## Description

A Firestore Backup in GCP, which provides a way to back up and restore Firestore databases to protect against data loss.
