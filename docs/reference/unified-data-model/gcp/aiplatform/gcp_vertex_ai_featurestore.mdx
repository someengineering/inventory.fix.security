# `gcp_vertex_ai_featurestore`

- Categories: Machine Learning & AI
- Service: aiplatform
- Provider Link: [Vertex AI Featurestore](https://cloud.google.com/docs)

## Description

A Featurestore is a fully managed Vertex AI service used to store, manage, and serve machine learning features at scale. It enables faster, scalable, and consistent access to features.
