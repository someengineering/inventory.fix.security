# `gcp_vertex_ai_custom_job`

- Categories: Machine Learning & AI, Compute
- Service: aiplatform
- Provider Link: [Vertex AI Custom Job](https://cloud.google.com/docs)

## Description

A Custom Job in Vertex AI allows users to run their own custom machine learning training or inference code on Google Cloud infrastructure.
