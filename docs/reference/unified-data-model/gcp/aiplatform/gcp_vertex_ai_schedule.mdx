# `gcp_vertex_ai_schedule`

- Categories: Machine Learning & AI
- Service: aiplatform
- Provider Link: [Vertex AI Schedule](https://cloud.google.com/docs)

## Description

A Schedule is a recurring execution of a task in Vertex AI, such as training or batch prediction jobs, triggered based on defined intervals.
