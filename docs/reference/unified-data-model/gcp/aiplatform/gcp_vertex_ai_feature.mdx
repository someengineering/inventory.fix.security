# `gcp_vertex_ai_feature`

- Categories: Machine Learning & AI
- Service: aiplatform
- Provider Link: [Vertex AI Feature](https://cloud.google.com/docs)

## Description

A Feature in Vertex AI is an individual data attribute stored in a Feature Store. It is used to improve the predictive accuracy of machine learning models by providing structured, consistent access to key data characteristics.
