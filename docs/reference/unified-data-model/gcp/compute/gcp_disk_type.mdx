# `gcp_disk_type`

- Categories: Storage
- Service: compute
- Provider Link: [Disk Type](https://cloud.google.com/compute/docs/disks#disk-types)

## Description

GCP Disk Type refers to the storage options available for virtual machine instances in Google Cloud Platform. It includes persistent disks like standard HDD, balanced SSD, and performance SSD, as well as local SSDs. These disk types offer different performance characteristics and price points, catering to various workload requirements and storage needs in cloud computing environments.
