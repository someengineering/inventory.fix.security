# `gcp_firewall_policy`

- Categories: Networking
- Service: compute
- Provider Link: [Firewall Policy](https://cloud.google.com/vpc/docs/firewall-policies)

## Description

GCP Firewall Policy is a network security feature in Google Cloud Platform. It defines rules to control incoming and outgoing traffic for virtual machine instances. The policy specifies which network connections are permitted or blocked based on criteria like IP addresses, ports, and protocols. It helps protect cloud resources from unauthorized access and potential security threats.
