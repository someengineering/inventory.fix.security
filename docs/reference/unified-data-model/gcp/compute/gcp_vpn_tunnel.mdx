# `gcp_vpn_tunnel`

- Categories: Networking
- Service: compute
- Provider Link: [VPN Tunnel](https://cloud.google.com/network-connectivity/docs/vpn/concepts/overview)

## Description

GCP VPN Tunnel is a secure connection service in Google Cloud Platform. It creates encrypted links between on-premises networks and Google Cloud resources over the public internet. This service helps organizations extend their private networks to the cloud, ensuring data transmission security and enabling hybrid cloud architectures while maintaining network isolation and control.
