# `gcp_vpn_gateway`

- Categories: Networking
- Service: compute
- Provider Link: [VPN Gateway](https://cloud.google.com/network-connectivity/docs/vpn/concepts/overview)

## Description

GCP VPN Gateway is a network component in Google Cloud Platform that establishes secure connections between on-premises networks and Google Cloud resources. It encrypts traffic over the public internet, creating a virtual private network. The gateway manages multiple tunnels, supports both static and dynamic routing, and provides secure access to cloud-based applications and data.
