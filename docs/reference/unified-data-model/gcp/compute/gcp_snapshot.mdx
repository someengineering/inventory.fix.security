# `gcp_snapshot`

- Categories: Storage
- Service: compute
- Provider Link: [Snapshot](https://cloud.google.com/compute/docs/disks/snapshots)

## Description

GCP Snapshot is a data backup feature for Google Cloud Platform. It creates point-in-time copies of persistent disks, providing a way to safeguard data and recover from system failures. Snapshots can be used to clone disks, transfer data between regions, or restore data to a previous state, offering protection against data loss and system disruptions.
