# `gcp_target_instance`

- Categories: Networking
- Service: compute
- Provider Link: [Target Instance](https://cloud.google.com/compute/docs/protocol-forwarding#targetinstances)

## Description

A GCP Target Instance is a network resource in Google Cloud Platform that receives and handles incoming traffic for specific IP addresses and ports. It acts as a proxy, forwarding requests to designated virtual machine instances, facilitating load balancing and traffic management. Target Instances can be used to implement various networking scenarios and improve application availability.
