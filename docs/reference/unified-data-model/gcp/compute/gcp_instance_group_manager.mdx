# `gcp_instance_group_manager`

- Categories: Compute
- Service: compute
- Provider Link: [Instance Group Manager](https://cloud.google.com/compute/docs/instance-groups/working-with-managed-instance-groups)

## Description

GCP Instance Group Manager is a Google Cloud Platform service that automates the creation, management, and scaling of virtual machine instances. It maintains a specified number of instances, replaces unhealthy ones, and adjusts the group size based on defined rules. The service handles load balancing, rolling updates, and auto-scaling to meet application demands.
