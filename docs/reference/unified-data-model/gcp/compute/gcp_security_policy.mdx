# `gcp_security_policy`

- Categories: Networking
- Service: compute
- Provider Link: [Security Policy](https://cloud.google.com/vpc/docs/security-policies)

## Description

GCP Security Policy is a set of rules and configurations that define access controls and network security for Google Cloud Platform resources. It manages firewall rules, network traffic, and user permissions across projects and organizations. This policy helps protect cloud infrastructure from unauthorized access and potential threats while enforcing compliance requirements.
