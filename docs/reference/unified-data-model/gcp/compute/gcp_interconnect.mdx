# `gcp_interconnect`

- Categories: Networking
- Service: compute
- Provider Link: [Interconnect](https://cloud.google.com/network-connectivity/docs/interconnect)

## Description

GCP Interconnect is a network connectivity service from Google Cloud Platform. It provides direct physical connections between on-premises networks and Google's network. This service offers dedicated bandwidth and reduced latency compared to public internet connections. It supports both layer 2 and layer 3 connectivity options, enhancing data transfer speeds and network security for organizations using Google Cloud services.
