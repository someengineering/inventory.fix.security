# `gcp_target_http_proxy`

- Categories: Networking
- Service: compute
- Provider Link: [Target HTTP Proxy](https://cloud.google.com/load-balancing/docs/target-proxies)

## Description

A GCP Target HTTP Proxy is a Google Cloud Platform resource that routes incoming HTTP requests to backend services. It acts as an intermediary between external clients and internal services, distributing traffic based on URL maps. Target HTTP Proxies work with global load balancers to manage and direct incoming HTTP traffic across multiple regions.
