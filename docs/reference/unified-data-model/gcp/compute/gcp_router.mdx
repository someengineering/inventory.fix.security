# `gcp_router`

- Categories: Networking
- Service: compute
- Provider Link: [Router](https://cloud.google.com/network-connectivity/docs/router)

## Description

GCP Router is a networking component in Google Cloud Platform that directs traffic between virtual networks and on-premises networks. It manages routing tables, implements BGP protocols, and facilitates communication between different subnets. GCP Router supports static and dynamic routing, enabling network administrators to control traffic flow and connect resources across various network environments.
