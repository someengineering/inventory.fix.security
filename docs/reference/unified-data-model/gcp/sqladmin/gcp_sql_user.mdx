# `gcp_sql_user`

- Categories: Database
- Service: sqladmin
- Provider Link: [SQL User](https://cloud.google.com/sql/docs/mysql/users)

## Description

A GCP SQL User is an account within Google Cloud Platform's Cloud SQL service that has permissions to access and manage databases. It can connect to databases, execute queries, modify data, and perform administrative tasks based on its assigned privileges. GCP SQL Users are created and managed by database administrators to control access to Cloud SQL instances.
