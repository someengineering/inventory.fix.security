# `aws_config_recorder`

- Categories: Management
- Service: config
- Provider Link: [Config Recorder](https://docs.aws.amazon.com/config/latest/developerguide/config-concepts.html#config-recorder)

## Description

AWS Config Recorder is a service that tracks and records configuration changes in AWS resources. It captures resource relationships and modifications, storing this information in a detailed history. This data can be used for compliance auditing, security analysis, and resource management. Config Recorder supports various AWS services and provides a foundation for configuration management.
