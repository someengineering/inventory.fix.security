# `aws_cloudfront_response_headers_policy`

- Categories: Networking
- Service: cloudfront
- Provider Link: [CloudFront Response Headers Policy](https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/adding-response-headers.html)

## Description

AWS CloudFront Response Headers Policy is a configuration that controls HTTP headers in responses sent from CloudFront to viewers. It lets users add, modify, or remove headers for improved security, caching, and content delivery. This policy can be applied to multiple distributions, simplifying header management across CloudFront's content delivery network.
