# `aws_cloudfront_field_level_encryption_profile`

- Categories: Networking
- Service: cloudfront
- Provider Link: [CloudFront Field Level Encryption Profile](https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/field-level-encryption.html)

## Description

AWS CloudFront Field Level Encryption Profile is a security feature that encrypts specific data fields in web forms before transmitting them to origin servers. It uses public key cryptography to protect sensitive information, ensuring that only authorized systems with the corresponding private key can decrypt the data. This adds an extra layer of protection for user-submitted data.
