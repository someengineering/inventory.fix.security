# `aws_apigateway_rest_api`

- Categories: Networking
- Service: apigateway
- Provider Link: [API Gateway REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html)

## Description

AWS API Gateway REST API is a service for creating, publishing, and managing APIs. It acts as an entry point for applications to access data, business logic, or functionality from backend services. The service handles tasks such as traffic management, authorization, access control, monitoring, and API version management for REST APIs.
