# `aws_efs_mount_target`

- Categories: Storage
- Service: efs
- Provider Link: [EFS Mount Target](https://docs.aws.amazon.com/efs/latest/ug/mounting-fs-mount-cmd-general.html)

## Description

An AWS EFS Mount Target is a network interface in a Virtual Private Cloud (VPC) subnet that connects to an Elastic File System (EFS). It provides an IP address for accessing the file system within the VPC. Mount targets act as endpoints for NFS clients to connect and mount EFS file systems on EC2 instances.
