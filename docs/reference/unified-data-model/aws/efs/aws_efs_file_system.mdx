# `aws_efs_file_system`

- Categories: Storage
- Service: efs
- Provider Link: [EFS File System](https://docs.aws.amazon.com/efs/latest/ug/)

## Description

AWS EFS (Elastic File System) is a cloud-based network file storage service for Amazon EC2 instances. It provides a shared file system that can be accessed by multiple EC2 instances simultaneously. EFS automatically scales storage capacity as files are added or removed, and users pay only for the storage they use.
