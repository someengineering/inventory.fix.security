# `aws_ecr_repository`

- Categories: Compute
- Service: ecr
- Provider Link: [ECR Repository](https://docs.aws.amazon.com/AmazonECR/latest/userguide/what-is-ecr.html)

## Description

AWS ECR (Elastic Container Registry) is a managed Docker container registry service. It stores, manages, and deploys container images for applications. ECR integrates with other AWS services, provides secure access control, and supports image scanning for vulnerabilities. Users can push, pull, and share Docker images within their AWS environment or with external parties.
