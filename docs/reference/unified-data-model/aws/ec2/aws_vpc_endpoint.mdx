# `aws_vpc_endpoint`

- Categories: Networking
- Service: ec2
- Provider Link: [VPC Endpoint](https://docs.aws.amazon.com/vpc/latest/privatelink/vpc-endpoints.html)

## Description

An AWS VPC Endpoint is a service that provides private connectivity between Amazon Virtual Private Clouds (VPCs) and supported AWS services. It routes traffic within the Amazon network, bypassing the public internet. VPC Endpoints improve security by keeping data within the AWS infrastructure and can reduce data transfer costs for certain services.
