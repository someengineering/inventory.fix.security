# `aws_vpc`

- Categories: Networking
- Service: ec2
- Provider Link: [VPC](https://docs.aws.amazon.com/vpc/)

## Description

AWS VPC (Amazon Virtual Private Cloud) is a network service that creates isolated cloud environments within AWS. It lets users define virtual networks, configure IP ranges, set up subnets, and manage network gateways. VPC provides control over network architecture, enhances security through access controls, and supports connectivity between AWS resources and on-premises infrastructure.
