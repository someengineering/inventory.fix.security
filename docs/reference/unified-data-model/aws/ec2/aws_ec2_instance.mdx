# `aws_ec2_instance`

- Categories: Compute
- Service: ec2
- Provider Link: [EC2 Instance](https://docs.aws.amazon.com/ec2/)

## Description

AWS EC2 Instance is a virtual server in Amazon's Elastic Compute Cloud. It provides computing capacity in the cloud, offering various configurations of CPU, memory, storage, and networking capacity. Users can launch instances with different operating systems, configure security and networking, and manage storage. EC2 instances support diverse workloads and applications, from web servers to databases.
