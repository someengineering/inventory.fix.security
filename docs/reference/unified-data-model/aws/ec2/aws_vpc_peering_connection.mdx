# `aws_vpc_peering_connection`

- Categories: Networking
- Service: ec2
- Provider Link: [VPC Peering Connection](https://docs.aws.amazon.com/vpc/latest/peering/what-is-vpc-peering.html)

## Description

AWS VPC Peering Connection is a networking feature that connects two Virtual Private Clouds (VPCs) within the same or different AWS accounts. It establishes direct network routing between VPCs, letting resources in each VPC communicate with each other using private IP addresses. This connection works across regions and improves network performance by bypassing the public internet for inter-VPC traffic.
