# `aws_ec2_route_table`

- Categories: Networking
- Service: ec2
- Provider Link: [EC2 Route Table](https://docs.aws.amazon.com/vpc/latest/userguide/VPC_Route_Tables.html)

## Description

An AWS EC2 Route Table is a networking component in Amazon Web Services that directs traffic between subnets within a Virtual Private Cloud (VPC) and to external networks. It contains rules, called routes, which determine where network traffic is sent based on its destination IP address. Route tables control both inbound and outbound traffic for associated subnets.
