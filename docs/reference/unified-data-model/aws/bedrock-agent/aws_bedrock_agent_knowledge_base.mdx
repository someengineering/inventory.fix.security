# `aws_bedrock_agent_knowledge_base`

- Categories: Machine Learning & AI
- Service: bedrock-agent
- Provider Link: [Bedrock Agent Knowledge Base](https://docs.aws.amazon.com/bedrock/latest/userguide/agents-knowledge-base.html)

## Description

AWS Bedrock Agent Knowledge Base is a feature that stores and manages information for AI agents. It provides a structured repository for data, documents, and facts that agents can access and use to answer questions, make decisions, and perform tasks. The knowledge base supports natural language queries and helps maintain context during interactions.
