# `aws_root_user`

- Categories: Access Control
- Service: iam
- Provider Link: [Root User](https://docs.aws.amazon.com/IAM/latest/UserGuide/id_root-user.html)

## Description

The AWS Root User is the primary account holder for an Amazon Web Services (AWS) account. It has full access to all AWS services and resources within the account. The Root User can create and manage other users, assign permissions, and make billing changes. It should be used sparingly for security reasons.
