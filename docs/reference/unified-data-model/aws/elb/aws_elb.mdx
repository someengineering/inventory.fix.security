# `aws_elb`

- Categories: Networking
- Service: elb
- Provider Link: [ELB](https://docs.aws.amazon.com/elasticloadbalancing/)

## Description

AWS Elastic Load Balancing (ELB) is a service that distributes incoming application traffic across multiple targets, such as EC2 instances, containers, and IP addresses. It automatically adjusts capacity based on traffic patterns, provides health checks for connected resources, and supports various load balancing algorithms to optimize application performance and availability in Amazon Web Services environments.
