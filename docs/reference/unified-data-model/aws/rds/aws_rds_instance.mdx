# `aws_rds_instance`

- Categories: Compute, Database
- Service: rds
- Provider Link: [RDS Instance](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/Overview.DBInstance.html)

## Description

AWS RDS Instance is a managed relational database service in Amazon Web Services. It provides a database instance with automated backups, software patching, and monitoring. Users can choose from various database engines, including MySQL, PostgreSQL, and Oracle. RDS handles routine database tasks, freeing developers to focus on application development and optimization.
