# `aws_acm_certificate`

- Categories: Access Control, Security
- Service: acm
- Provider Link: [ACM Certificate](https://docs.aws.amazon.com/acm/latest/userguide/acm-overview.html)

## Description

AWS ACM Certificate is a digital credential issued by Amazon Web Services Certificate Manager. It authenticates the identity of websites and secures connections between clients and servers. ACM Certificates encrypt data in transit, prevent unauthorized access, and establish trust for online services. They support various AWS services and can be deployed across multiple regions.
