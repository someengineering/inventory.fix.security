# `azure_monitor_log_profile`

- Categories: Management
- Service: monitor
- Provider Link: [Monitor Log Profile](https://learn.microsoft.com/en-us/azure/azure-monitor/essentials/activity-log)

## Description

Azure Monitor Log Profile is a configuration setting that defines how activity logs are collected and stored in Azure. It specifies which log categories to capture, the retention period for logs, and the destination for log data. This profile helps organizations manage their Azure resource monitoring and maintain compliance with data retention policies.
