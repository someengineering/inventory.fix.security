# `azure_monitor_activity_log_alert`

- Categories: Management
- Service: monitor
- Provider Link: [Monitor Activity Log Alert](https://docs.microsoft.com/en-us/azure/azure-monitor/alerts/alerts-activity-log)

## Description

Azure Monitor Activity Log Alert is a service that monitors and notifies users about specific events or changes in Azure resources. It tracks operations performed on Azure services, including resource creation, modification, or deletion. Users can set up custom alerts based on defined criteria and receive notifications through various channels when those conditions are met.
