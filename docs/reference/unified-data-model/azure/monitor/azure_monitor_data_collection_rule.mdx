# `azure_monitor_data_collection_rule`

- Categories: Management
- Service: monitor
- Provider Link: [Monitor Data Collection Rule](https://learn.microsoft.com/en-us/azure/azure-monitor/essentials/data-collection-rule-overview)

## Description

Azure Monitor Data Collection Rule is a configuration resource that defines how monitoring data is collected and processed in Azure. It specifies the data sources, collection settings, and destinations for log and metric data. Data Collection Rules help manage and organize monitoring across multiple resources, ensuring consistent data collection and routing within Azure environments.
