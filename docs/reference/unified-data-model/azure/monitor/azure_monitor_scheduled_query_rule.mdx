# `azure_monitor_scheduled_query_rule`

- Categories: Management
- Service: monitor
- Provider Link: [Monitor Scheduled Query Rule](https://learn.microsoft.com/en-us/azure/azure-monitor/alerts/alerts-scheduled-query)

## Description

Azure Monitor Scheduled Query Rule is a feature that executes predefined queries on log data at specified intervals. It evaluates the query results against set thresholds and triggers alerts when conditions are met. This tool helps users monitor their Azure resources and applications, detect issues, and respond to potential problems based on collected telemetry data.
