# `azure_sql_server_job_agent`

- Categories: Database
- Service: azure-sql
- Provider Link: [SQL Server Job Agent](https://learn.microsoft.com/en-us/azure/azure-sql/database/job-agent-overview)

## Description

Azure SQL Server Job Agent is a component of Azure SQL Managed Instance that automates and schedules database maintenance tasks. It executes jobs containing Transact-SQL scripts across multiple databases, performs routine operations, and manages backups. The agent supports recurring schedules, monitors job status, and provides notifications for successful or failed job runs.
