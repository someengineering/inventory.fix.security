# `azure_sql_server_ad_administrator`

- Categories: Database
- Service: azure-sql
- Provider Link: [SQL Server Ad Administrator](https://learn.microsoft.com/en-us/azure/azure-sql/database/authentication-aad-configure?tabs=azure-powershell)

## Description

Azure SQL Server AD Administrator is a role that integrates Azure Active Directory authentication with SQL Server. It manages access control for SQL databases using Azure AD credentials, allowing organizations to centralize user management and enforce multi-factor authentication. This role simplifies security administration by applying Azure AD policies and permissions to SQL Server resources.
