# `azure_sql_server_managed_instance_failover_group`

- Categories: Database
- Service: azure-sql
- Provider Link: [SQL Server Managed Instance Failover Group](https://learn.microsoft.com/en-us/azure/azure-sql/managed-instance/auto-failover-group-overview)

## Description

Azure SQL Server Managed Instance Failover Group is a feature that provides database redundancy and high availability across multiple regions. It automatically replicates databases from a primary managed instance to a secondary instance in a different region. During outages or disasters, it initiates failover to the secondary instance, ensuring continuous data access and minimizing downtime for applications.
