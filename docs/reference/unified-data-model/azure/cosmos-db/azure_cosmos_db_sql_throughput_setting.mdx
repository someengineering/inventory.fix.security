# `azure_cosmos_db_sql_throughput_setting`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB SQL Throughput Setting](https://learn.microsoft.com/en-us/azure/cosmos-db/sql/how-to-provision-container-throughput)

## Description

Azure Cosmos DB SQL Throughput Setting controls the performance and capacity of a Cosmos DB container or database. It determines the amount of resources allocated for read and write operations, measured in Request Units per second (RU/s). Users can adjust this setting to meet their application's needs, balancing performance requirements with cost considerations.
