# `azure_cosmos_db_account`

- Categories: Compute, Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Account](https://learn.microsoft.com/en-us/azure/cosmos-db/)

## Description

Azure Cosmos DB Account is a cloud-based database service offered by Microsoft Azure. It provides a globally distributed, multi-model database platform for storing and managing data. Users can create and manage databases, containers, and items within the account. It supports multiple APIs, including SQL, MongoDB, Cassandra, Gremlin, and Table, offering flexibility for various application needs.
