# `azure_cosmos_db_cassandra_cluster_public_status`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Cassandra Cluster Public Status](https://learn.microsoft.com/en-us/azure/cosmos-db/cassandra/)

## Description

Azure Cosmos DB Cassandra Cluster Public Status is a monitoring feature that displays the current operational condition of Cassandra clusters within Azure Cosmos DB. It provides real-time information about cluster health, availability, and performance metrics. Users can check this status to assess service reliability, identify potential issues, and make informed decisions about their database operations.
