# `azure_cosmos_db_postgresql_cluster`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB PostgreSQL Cluster](https://learn.microsoft.com/en-us/azure/cosmos-db/postgresql/)

## Description

Azure Cosmos DB PostgreSQL Cluster is a managed database service that provides PostgreSQL compatibility within the Azure cloud environment. It offers distributed data storage and processing capabilities, supporting multi-region deployments and automatic scaling. The service handles database administration tasks, including backups, updates, and security, while maintaining PostgreSQL compatibility for existing applications and tools.
