# `azure_cosmos_db_postgresql_cluster_role`

- Categories: Access Control
- Service: cosmos-db
- Provider Link: [Cosmos DB PostgreSQL Cluster Role](https://learn.microsoft.com/en-us/azure/cosmos-db/postgresql/concepts-roles)

## Description

Azure Cosmos DB PostgreSQL Cluster Role represents a PostgreSQL database instance within an Azure Cosmos DB cluster. It provides PostgreSQL-compatible database services with distributed capabilities. This role handles data storage, query processing, and transaction management while offering compatibility with existing PostgreSQL tools and applications in a distributed environment.
