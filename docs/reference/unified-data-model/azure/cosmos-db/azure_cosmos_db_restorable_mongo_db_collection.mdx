# `azure_cosmos_db_restorable_mongo_db_collection`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Restorable Mongo DB Collection](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/restorable-mongodb-resources)

## Description

Azure Cosmos DB Restorable MongoDB Collection is a feature that provides point-in-time recovery for MongoDB databases in Azure Cosmos DB. It creates backups of collections at regular intervals, letting users restore data to a specific timestamp within the retention period. This helps recover from accidental deletions, modifications, or application errors.
