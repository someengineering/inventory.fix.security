# `azure_cosmos_db_restorable_mongo_db_database`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Restorable Mongo DB Database](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/restorable-mongodb-resources)

## Description

Azure Cosmos DB Restorable MongoDB Database is a feature that provides point-in-time recovery for MongoDB databases within Azure Cosmos DB. It creates periodic backups of your database, letting you restore to any moment within the retention period. This helps protect against data loss, corruption, or accidental changes, ensuring data continuity and recovery options.
