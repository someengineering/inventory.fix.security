# `azure_cosmos_db_restorable_account`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Restorable Account](https://learn.microsoft.com/en-us/azure/cosmos-db/configure-periodic-backup-restore)

## Description

Azure Cosmos DB Restorable Account is a feature that provides data recovery capabilities for Azure Cosmos DB databases. It creates periodic backups of your database, letting you restore your account to a specific point in time within the last 30 days. This helps protect against accidental deletions, modifications, or other data loss scenarios.
