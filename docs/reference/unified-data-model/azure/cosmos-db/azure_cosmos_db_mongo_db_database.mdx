# `azure_cosmos_db_mongo_db_database`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Mongo DB Database](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/)

## Description

Azure Cosmos DB MongoDB Database is a cloud-based service that provides MongoDB API compatibility within Microsoft's Azure ecosystem. It offers document storage and querying capabilities, supporting MongoDB operations and drivers. Users can create, read, update, and delete data using familiar MongoDB syntax while benefiting from Azure's global distribution and consistency options.
