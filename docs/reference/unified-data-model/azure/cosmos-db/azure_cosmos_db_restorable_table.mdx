# `azure_cosmos_db_restorable_table`

- Categories: Database
- Service: cosmos-db
- Provider Link: [Cosmos DB Restorable Table](https://learn.microsoft.com/en-us/azure/cosmos-db/table/restorable-table-accounts)

## Description

Azure Cosmos DB Restorable Table is a feature that provides point-in-time recovery for Azure Cosmos DB table data. It creates periodic backups of table data, allowing users to restore tables to a specific timestamp within the retention period. This functionality helps protect against accidental deletions, modifications, or corruption of data in Azure Cosmos DB tables.
