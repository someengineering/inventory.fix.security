# `azure_compute_virtual_machine_scale_set`

- Categories: Compute, Management
- Service: compute
- Provider Link: [Compute Virtual Machine Scale Set](https://learn.microsoft.com/en-us/azure/virtual-machine-scale-sets/)

## Description

Azure Compute Virtual Machine Scale Set is a service that manages and scales multiple identical virtual machines. It automates the creation, updating, and deletion of VMs based on demand or schedules. The service distributes traffic, handles load balancing, and integrates with Azure monitoring tools to maintain performance and availability across the scaled instances.
