# `azure_compute_proximity_placement_group`

- Categories: Compute
- Service: compute
- Provider Link: [Compute Proximity Placement Group](https://learn.microsoft.com/en-us/azure/virtual-machines/co-location)

## Description

Azure Compute Proximity Placement Group is a feature that groups virtual machines within a single Azure datacenter. It minimizes network latency between VMs by placing them physically close to each other. This configuration improves performance for applications that require low-latency communication between components, such as high-performance computing or tightly coupled distributed systems.
