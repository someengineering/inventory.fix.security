# `azure_compute_virtual_machine`

- Categories: Compute
- Service: compute
- Provider Link: [Compute Virtual Machine](https://learn.microsoft.com/en-us/azure/virtual-machines/)

## Description

Azure Compute Virtual Machine is a cloud service that provides on-demand, configurable computing resources. It offers Windows or Linux-based virtual machines that run on Microsoft's infrastructure. Users can select from various machine sizes and types, install custom software, and manage their VMs remotely. These VMs support diverse workloads, from development and testing to running applications.
