# `azure_compute_disk_type_pricing`

- Categories: Other
- Service: compute
- Provider Link: [Compute Disk Type Pricing](https://azure.microsoft.com/en-us/pricing/details/managed-disks/)

## Description

Azure Compute Disk Type Pricing refers to the cost structure for different storage options available for virtual machines in Microsoft Azure. It outlines the prices for various disk types, including Standard HDD, Standard SSD, and Premium SSD, based on their performance characteristics, capacity, and usage. This pricing model helps users select appropriate storage solutions for their workloads.
