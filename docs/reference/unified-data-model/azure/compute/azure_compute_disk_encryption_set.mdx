# `azure_compute_disk_encryption_set`

- Categories: Management
- Service: compute
- Provider Link: [Compute Disk Encryption Set](https://learn.microsoft.com/en-us/azure/virtual-machines/disk-encryption-sets)

## Description

Azure Compute Disk Encryption Set is a resource that manages encryption keys for Azure virtual machine disks. It provides centralized control over key management and encryption settings for multiple disks. Users can apply consistent encryption policies across their VM infrastructure, enhancing data protection and compliance with security standards.
