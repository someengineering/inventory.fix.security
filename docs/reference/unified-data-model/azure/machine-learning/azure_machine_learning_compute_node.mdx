# `azure_machine_learning_compute_node`

- Categories: Compute
- Service: machine-learning
- Provider Link: [Machine Learning Compute Node](https://learn.microsoft.com/en-us/azure/machine-learning/concept-compute-target)

## Description

Azure Machine Learning Compute Node is a managed compute resource for running machine learning workloads. It provides a dedicated virtual machine environment for training models, conducting experiments, and deploying solutions. Users can configure the node's specifications, including CPU, GPU, and memory, to match their computational needs. It integrates with other Azure ML services for data processing and model management.
