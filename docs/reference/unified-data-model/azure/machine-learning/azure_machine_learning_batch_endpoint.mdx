# `azure_machine_learning_batch_endpoint`

- Categories: Machine Learning & AI
- Service: machine-learning
- Provider Link: [Machine Learning Batch Endpoint](https://learn.microsoft.com/en-us/azure/machine-learning/how-to-use-batch-endpoint)

## Description

Azure Machine Learning Batch Endpoint is a feature for processing large volumes of data asynchronously. It provides a consistent interface for running machine learning models on batches of data, managing compute resources, and retrieving results. Users can schedule jobs, monitor progress, and access outputs through API calls or the Azure portal.
