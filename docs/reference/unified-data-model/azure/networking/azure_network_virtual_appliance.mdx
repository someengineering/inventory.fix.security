# `azure_network_virtual_appliance`

- Categories: Networking
- Service: networking
- Provider Link: [Network Virtual Appliance](https://learn.microsoft.com/en-us/azure/virtual-network/virtual-network-vnet-plan-design-arm#network-virtual-appliances)

## Description

Azure Network Virtual Appliance is a network security service in Microsoft Azure. It functions as a virtual firewall, routing traffic between subnets and enforcing security policies. The appliance inspects network packets, filters traffic, and protects against threats. It can be deployed in virtual networks to control data flow and implement network security rules.
