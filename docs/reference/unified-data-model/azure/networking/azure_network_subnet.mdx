# `azure_network_subnet`

- Categories: Networking
- Service: networking
- Provider Link: [Network Subnet](https://learn.microsoft.com/en-us/azure/virtual-network/virtual-network-manage-subnet)

## Description

Azure Network Subnet is a logical subdivision of an Azure virtual network. It segments the network address space, organizing resources and controlling traffic flow. Subnets define IP address ranges for devices and services, enabling network isolation and security. They facilitate communication between resources within the same subnet and manage access to other subnets or external networks.
