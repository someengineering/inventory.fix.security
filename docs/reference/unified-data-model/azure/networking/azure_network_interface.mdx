# `azure_network_interface`

- Categories: Networking
- Service: networking
- Provider Link: [Network Interface](https://learn.microsoft.com/en-us/azure/virtual-network/virtual-network-network-interface)

## Description

An Azure Network Interface is a virtual network adapter that connects Azure virtual machines to the internet, Azure virtual networks, and on-premises networks. It assigns IP addresses, network security group rules, and load balancing settings to VMs. Network Interfaces can be attached to or detached from VMs and configured with multiple IP configurations.
