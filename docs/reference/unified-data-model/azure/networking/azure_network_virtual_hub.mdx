# `azure_network_virtual_hub`

- Categories: Networking
- Service: networking
- Provider Link: [Network Virtual Hub](https://learn.microsoft.com/en-us/azure/virtual-wan/virtual-wan-about)

## Description

Azure Network Virtual Hub is a managed networking service in Microsoft Azure. It serves as a central connection point for virtual networks and on-premises networks. The hub facilitates secure communication between connected networks, implements routing, and integrates with other Azure networking services. It supports site-to-site VPN, ExpressRoute, and point-to-site VPN connections.
