# `azure_network_ddos_protection_plan`

- Categories: Networking
- Service: networking
- Provider Link: [Network DDoS Protection Plan](https://learn.microsoft.com/en-us/azure/ddos-protection/ddos-protection-overview)

## Description

Azure Network DDoS Protection Plan is a service that defends Azure resources against Distributed Denial of Service attacks. It monitors network traffic, detects potential threats, and automatically mitigates them. The plan provides real-time attack metrics, alerting, and logging. It offers protection for virtual networks, public IP addresses, and application gateways without requiring changes to applications.
