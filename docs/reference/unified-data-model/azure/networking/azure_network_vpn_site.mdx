# `azure_network_vpn_site`

- Categories: Networking
- Service: networking
- Provider Link: [Network VPN Site](https://learn.microsoft.com/en-us/azure/vpn-gateway/vpn-gateway-about-vpn-gateway)

## Description

Azure Network VPN Site is a service that creates secure connections between on-premises networks and Azure virtual networks. It establishes encrypted tunnels over the internet, facilitating data transfer and resource access across locations. This service supports both site-to-site and point-to-site configurations, providing remote users with access to Azure resources while maintaining network security.
