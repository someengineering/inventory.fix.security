# `azure_resource_group`

- Categories: Access Control, Management
- Service: resources
- Provider Link: [Resource Group](https://learn.microsoft.com/en-us/azure/azure-resource-manager/management/manage-resource-groups-portal)

## Description

An Azure Resource Group is a container for organizing and managing related Azure resources. It serves as a logical unit for grouping services, applications, and infrastructure components within a single Azure subscription. Resource Groups help users control access, track costs, and apply policies across multiple resources, simplifying administration and deployment of cloud-based solutions.
