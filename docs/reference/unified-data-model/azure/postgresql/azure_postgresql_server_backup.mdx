# `azure_postgresql_server_backup`

- Categories: Database
- Service: postgresql
- Provider Link: [PostgreSQL Server Backup](https://learn.microsoft.com/en-us/azure/postgresql/flexible-server/concepts-backup-restore)

## Description

Azure PostgreSQL Server Backup is a feature in Azure Database for PostgreSQL that creates backups of your database automatically. It stores these backups in geo-redundant storage for data protection. The service retains backups for a specified period and supports point-in-time recovery, letting users restore their database to a previous state within the retention period.
