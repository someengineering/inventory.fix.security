# `azure_mysql_server_database`

- Categories: Database
- Service: mysql
- Provider Link: [MySQL Server Database](https://learn.microsoft.com/en-us/azure/mysql/)

## Description

Azure MySQL Server Database is a managed relational database service on Microsoft's cloud platform. It provides MySQL-compatible database capabilities, handling infrastructure management, backups, and security. Users can create, scale, and operate MySQL databases in the cloud, with options for high availability and automatic updates while maintaining compatibility with existing MySQL applications and tools.
