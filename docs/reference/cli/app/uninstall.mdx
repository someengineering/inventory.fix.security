# `app uninstall` Command

The `app uninstall` command uninstalls an app.

## Usage

```bash
app uninstall <app_name>
```

### Parameters

| Parameter  | Description             | Required? | Default Value |
| ---------- | ----------------------- | --------- | ------------- |
| `app_name` | Infrastructure app name | ✔️        |               |

## Example

```bash
> app uninstall tagvalidator
# highlight-start
App tagvalidator uninstalled
# highlight-end
```
