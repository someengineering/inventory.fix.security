# `app info` Command

The `app info` command shows information about an app.

## Usage

```bash
app info <app_name>
```

### Parameters

| Parameter  | Description             | Required? | Default Value |
| ---------- | ----------------------- | --------- | ------------- |
| `app_name` | Infrastructure app name | ✔️        |               |

## Example

```bash
> app info tagvalidator
# highlight-start
name: tagvalidator
description: Tag Validator app for Fix Inventory.
url: https://fixinventory.org/
version: 1.0.0
# highlight-end
```
