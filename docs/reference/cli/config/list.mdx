# `config list` Command

The `config list` command lists all configuration identifiers.

## Usage

```bash
config list
```

## Examples

```bash
> configs list
# highlight-start
​fixinventory.core
​fixinventory.worker.1
​fixinventory.worker.2
​fixinventory.metrics
# highlight-end
```
