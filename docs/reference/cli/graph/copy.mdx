# `graph copy` Command

The `graph copy` command copies a Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshot.

## Usage

```bash
graph copy [--force] [from_graph_name] <to_graph_name>
```

## Example

```bash
> graph copy fix fix_backup
# highlight-start
Graph fix copied to fix_backup.
# highlight-end
```
