# `graph snapshot` Command

The `graph snapshot` command creates a Fix Inventory Rewind [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshot.

It is an alias for the `graph copy` command with added `snapshot-` prefix and label/timestamp suffix.

## Usage

```bash
graph snapshot [from_graph_name] <snapshot_label>
```

## Example

```bash
> graph snapshot fix daily
# highlight-start
Graph fix snapshoted to snapshot-fix-daily-20230619T121940Z.
# highlight-end
```
